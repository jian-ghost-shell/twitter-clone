# Twitter Clone - Future Features (P1)

P1 = Priority 1 features to implement later
P0 = Already completed ✅

- [x] Post tweet (text only)
- [x] View feed (all tweets)

### Phase 2: User Authentication
- [x] NextAuth.js setup

### Phase 3: Interactions
